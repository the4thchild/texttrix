Welcome to Text Trix!  We're just getting underway here.
Please try us out!

System Reqs:
-Java Runtime Environment or Software Development Kit 1.3+,
freely available from Sun (http://java.sun.com/j2se/1.4)

Instructions for use (see http://texttrix.sourceforge.net/downloads.html
for more details):
-unzip TextTrix-x.y.z.zip
-Windows users can next simply double-click on TextTrix-x.y.z.jar; 
Linux users can do the same if the application/x-jar MIME type is 
supported and issues the command, "java -jar", on the file

-to start Text Trix via a command line, follow the next instructions:
-move into the TextTrix folder:
(Linux)
$ cd [directory where unzipped]/TextTrix-x.y.z
(Windows MS-DOS/command prompt)
> cd [directory where unzipped]\TextTrix-x.y.z
-start the TextTrix-x.y.z.jar file:
(Linux or Windows) $ java -jar TextTrix-x.y.z.jar

Be a text-ie techie!

Extra Hard Returns Remover

Removes all hard returns, except between
multiply-spaced lines or in lists

-to write a list without
losing its hard returns, keep all the elements
in a given block single-spaced, and use dashes or
asterisks at the start of each line after the first;
note that indents with tabs are a-ok

-text enclosed in <pre></pre> tags will be preserved;
the tags should each be placed on its own line and
will be removed

-lines beginning with ">" or "<", such
as those in email replies, will be removed along with
any spaces surrounding them; "---Original Message----"
will be added before the region, and "-------------" will
come afterward

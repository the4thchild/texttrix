Welcome to Text Trix™!

Text Trix is an open-source, cross-platform text editor. Its goal is to make text file navigation easier for coders.

A bug fix and minor enhancements release to Text Trix. This release features fixes to startup on Vista and build errors. Enhancements include Line Dance improvements and text drag-n-drop.

New in v.0.7.1rc1:

-Line Dance updates: names line from selected text; fixed table update when shifting between tab groups.
-Word Find: selects text when entering the box.
-Line Find (newly updated): A box that takes you directly to the given line number. Right-click on the line number in the status bar to record that number in Line Find.
-Source code repository: first release after migrating from CVS to Subversion repository.
-Group unindent: fixed on Linux.
-Drag 'n' Drop text.
-Code cleanup and bug fixes, including some new icons, script updates.
